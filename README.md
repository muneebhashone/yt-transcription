# yt-transcription

A CLI tool to fetch YouTube video transcripts using YouTube's internal Innertube API. Zero external dependencies ‚Äî runs on [Bun](https://bun.sh) with native `fetch`.

## How it works

1. **Fetches the video page** to extract the `INNERTUBE_API_KEY` and session cookies
2. **Calls the Innertube player endpoint** with an Android client context ‚Äî this returns caption URLs without PoToken restrictions
3. **Fetches and parses the caption XML** into structured transcript segments

No API keys required. No headless browsers. No third-party libraries.

## Prerequisites

- [Bun](https://bun.sh) runtime

## Setup

```bash
git clone https://github.com/muneebhashone/yt-transcription.git
cd yt-transcription
bun install
```

## Usage

```bash
bun run index.ts <youtube-url> [options]
```

### Options

| Flag | Description | Default |
|------|-------------|---------|
| `--format <fmt>` | Output format: `txt`, `srt`, `json` | `txt` |
| `--lang <code>` | Language code (e.g., `en`, `es`, `fr`) | auto-detect |
| `--output <path>` | Output file path | `<videoId>.<format>` |
| `-h, --help` | Show help message | |

### Examples

```bash
# Basic usage ‚Äî saves transcript as txt
bun run index.ts https://www.youtube.com/watch?v=dQw4w9WgXcQ

# Export as SRT subtitles
bun run index.ts dQw4w9WgXcQ --format srt

# Fetch Spanish transcript and save to custom path
bun run index.ts https://youtu.be/dQw4w9WgXcQ --lang es --output transcript.txt

# Export as structured JSON
bun run index.ts dQw4w9WgXcQ --format json
```

### Supported URL formats

- `https://www.youtube.com/watch?v=VIDEO_ID`
- `https://youtu.be/VIDEO_ID`
- `https://www.youtube.com/shorts/VIDEO_ID`
- `https://www.youtube.com/embed/VIDEO_ID`
- Raw 11-character video ID

## Output formats

**txt** ‚Äî Clean, flowing plain text transcript

**srt** ‚Äî Standard SRT subtitle format with timestamps

```
1
00:00:18,640 --> 00:00:21,880
We're no strangers to love

2
00:00:22,640 --> 00:00:26,960
You know the rules and so do I
```

**json** ‚Äî Structured data with video metadata and timed segments

```json
{
  "videoId": "dQw4w9WgXcQ",
  "title": "Rick Astley - Never Gonna Give You Up",
  "language": "English",
  "isAutoGenerated": false,
  "segments": [
    { "text": "We're no strangers to love", "start": 18.64, "duration": 3.24 }
  ]
}
```

## Limitations

- Uses YouTube's undocumented Innertube API ‚Äî may break if YouTube changes their internals
- May be rate-limited or IP-blocked with heavy usage
- Only works with videos that have captions (manual or auto-generated)
